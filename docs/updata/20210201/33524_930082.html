Insurance System and Ship Recovery Rules (Updated as of 2/2)

2021-02-01

Insurance System Introduction
 
 
1. Introduction
Players can use the Insurance Points (IP) to recover the destroyed ships on their own, including the fittings and the rigs. The new Insurance system is improved from the previous Beginner's Insurance. Beginner's Insurance Vouchers will be converted to Insurance Points accordingly. When you recover the ship, the required IP is related to the estimated value of the ship, the items and PLEX, and the IP you’ve deposited. 
 
2. Recovery Rules
a) All the ships can be recovered by insurance (not Capsules). 
b) Only ships destroyed in the past 60 days can be recovered. 
c) The modules are optional; if they are chosen, then it will cost more IP.
d) Neither the dropped items, nor the items in the cargo hold, can be recovered with insurance. 
e) Some special items cannot be recovered. 
 
3. How to get IP
a) You can purchase IP with PLEX in the [Combat Log] interface. 
b) You can obtain some through daily login rewards. 
c) You can obtain some in the Basic Tutorials. 
 
4. Depositing IP
Players can deposit IP. When the ship is destroyed, there will be a discount of up to 70% based on the amount deposited. 
 
 
*Calculation:
Deposited IP = X, IP Cost of Recovering a Ship = Y
If X>=Y*0.3, the acutal IP Cost is Y*30%
If X<Y*0.3, the acutal IP Cost is (Y-X/0.3) + X
 
As you can see, you can save a lot simply by having enough IP deposited in advance!
More details about this can be found when purchasing IP.
 
Note: 
a) The ship hull is covered by default when using insurance, while the fittings and the rigs are optional. 
b) The Beginner's Insurance Voucher will be converted to Insurance Points at a 1:1 ratio. 
c) If there is no available estimated value of the ship in the market, it will be estimated by the cost of the reverse engineering or manufacturing materials. 
 
 
Ship Recovery Rules (Updated as of 2/2)
 
1. For ships that are destroyed before the new insurance system has been implemented:
a) They cannot be recovered with IP.
b) If the ship is destroyed due to a bug and before the new insurance system has been implemented, please refer to the previous Ship Recovery Rules.
 
2. For ships that are destroyed after the new insurance system is available in the game:
a) Please recover the ship using IP (for both PvE and PvP).
b) If you think that your ship is destroyed due to a bug, please recover the ship using IP first. Then you can contact customer service and send the details and the evidence of the bug. We will check further to see if your case is eligible for an IP refund.
c) If you think that your ship is destroyed due to a performance issue, please recover the ship by IP first. Then you can contact the customer service, and send the details and the evidence of the problem. We will check further if the situation is eligible for the IP refund.
 
Please note that for performance issues, there will be a 30-day cooldown for both PvE and PvP compensation. 
 
FAQ
Q: Can I use the fast recovery system for PvE after the new insurance system is available? 
A: If you have recovered your ship using IP, it cannot be recovered again in the fast PvE recovery system. Once a ship is recovered, it cannot be recovered in any other form of ship recovery. Our system can detect whether or not the ship was previously restored. 
 
The automatic recovery system for PvE will be phased out after a short period after the insurance system is implemented. 
 
Q: I restored my ship, but I forgot to recover the fittings. Can I contact customer service to manually recover my fittings?
A: If your ship is not destroyed due to a confirmed bug, please recover the ship using IP. If you forget to recover the fittings, then we are afraid that we currently cannot help recover them, as that is caused by an action made by the player. Please make sure you double check the recovery options when using IP.
 
Q: I recovered my ship with IP today. Will there be a cooldown for my next ship recovery using IP?
A: Only a cooldown applies to the fast recovery system for PvE and the manual ship recovery. The new insurance system has no cooldown for recovering your ship. 
 
However, the ships recovered using IP will not trigger a cooldown for the fast recovery system and the manual ship recovery.
