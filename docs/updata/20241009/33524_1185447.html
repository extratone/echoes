Patch Notes 1009

2024-10-09

To New Eden Capsuleers,

EVE Echoes will undergo maintenance on 10/9/2024 at 08:00 (UTC+0) and will take approximately 120 minutes to complete. During this time, players will be unable to log in to the game. We kindly ask for your understanding and patience during the maintenance.

 

Optimizations

The Space Scan and Base Scan features now only require abnormal Scanning Authorization and no longer need Citadel Management authority.

The description interface of the Lucky Chest will show the current cumulative number of chests opened (the count will reset to 0 after obtaining an important reward).

The effective range of the Warp and Jump fleet commands has been expanded from nearby fleet members to all members within the same system.

 

Bug Fixes

Fixed an issue where the guest list might not be displayed in the Station.

Fixed an issue where players might fail to enter the Corporation Clash gameplay successfully.

Fixed an issue where only the latest created squad would be displayed when inviting players into a fleet.

Fixed an issue where the Home Screen might show wrong buttons like Quit or Jump after undocking from the Station.

Fixed the numerical precision issues in the interfaces of Events, Reprocess, Compression, and NPC attributes.

Fixed an issue where Industrial Modules were categorized as Low Slots in the Manufacture interface.

Fixed an issue where the Sort by Distance feature in the acquire interface might not be functional.

Fixed a bug where the cargo interface might become unresponsive in rare circumstances.

Fixed an issue where the navigation feature in the Planetary Production interface might not respond when used in the Station.

Fixed a bug where the module replacement interface would automatically close.

Fixed a bug where the Kinetic Burst implant was incorrectly increasing hit chance.

 

Engine Update Reminder

After the release of the new engine, most Capsuleers have downloaded the new client to experience it and have continuously provided feedback to help fix bugs. We appreciate the support and love from all Capsuleers.

With the continuous updates, the new version client has now become stable. To improve development and testing efficiency, we plan to stop supporting the old version client login after the maintenance on October 9th and will send reward emails to all servers. At that time, we will focus more on optimizing the performance and fixing bugs of the new version.

 

Capsuleers who have not yet updated can follow the instructions on the event page to download and install version 5.2.2 or higher from the app store. After logging in, you can claim your update reward.

 

Important Notice:

Engine version 5.2.2 and above do not support 32-bit Android operating systems. It is recommended to use a 64-bit operating system with Android 9 or higher. If the game can no longer run after installation, please choose another device.
