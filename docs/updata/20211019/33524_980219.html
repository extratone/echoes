Patch Notes - October 20

2021-10-19

EVE Echoes will undergo maintenance on 10/20 at 08:00 (UTC+0) and will take approximately 120 minutes to complete. During this time, players will be unable to log in to the game.

October 20 Version Patch Notes

Virtual Test: Interference — Carriers Reporting for Duty!

Event Time: 10/20 after server maintenance – 10/27 after server maintenance

Virtual Test: Interference progresses, and we see a huge breakthrough in capital ship technology. After the server maintenance on 10/20, four new carriers will debut in Virtual Test: Interference: Chimera, Nidhoggur, Archon, and Thanatos.

Pilots will be able to choose to pilot these four carriers in Virtual Test: Interference battles and try out the unique carrier fighter mechanic, which will no doubt introduce players to some thrilling combat between carriers.

Check the in-game event rules for more details.

Crisis Approaches: The Tide Rises — Unveiling the Ship Graveyard

Event Time: 10/24 at 00:00 – 11/3 at 00:00 UTC+0
Reward Claim Deadline: 11/11 at 00:00 UTC+0

The adversary from Void Space fans the flames of war to every corner of New Eden. To protect New Eden, CONCORD and the four empires call out to pilots to venture into the Nihilus Void Space to eliminate the "Interferer" fleet. What secrets do the "Interferer" shipwrecks hold?

Defeat the fleet to progress through the event, which is shown on the "Ship Graveyard" event page. With each third of the event completed, combat difficulty will increase. Every time the event reaches a new stage, all pilots will receive great rewards. All unlocked rewards will remain claimable until 11/11 at 00:00 UTC+0.

Check the in-game event rules for more details.

The End of the Ship Data Test Manual Event Draws Near
As Virtual Test: Interference approaches its conclusion, the Ship Data Test Manual event will end 10/21 at 00:00. Pilots, please claim your unclaimed rewards in due time.

Optimizations
1. Adjusted the premium factor of the material valuation formula.
2. Adjusted the function of the Report button.
3. Changed part of the Dark Halo Core's description: Warp Scramble Range > Disruption Field Range.
4. After fitting a module, fitting another will record the listed location of the last one.
5. Added: a close button to the search screen.
6. You can now switch the styles of the bottom panel of the main interface module button.
7. Added carriers to Virtual Test (dreadnoughts are still available).
8. Changed the non-player dreadnoughts to carriers in the third test to test the combat capabilities of carriers when not controlled by a player.

Bug Fixes
1. Fixed the issue where Nihilus Space generation would fail under some circumstances.
2. Fixed the issue where the mission reward screen wouldn't disappear after the rewards were claimed.
