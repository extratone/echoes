BALANCE UPDATE TEST SERVER ANNOUNCEMENT

2022-04-08

Dear Pilots, 
The EVE Echoes Test Server for the balance update will be open at 8:00 am (UTC+0), April 9th and will close at 10am (UTC+0), April 12th. To ensure players can continue to play on the Normal Server during the test, the test server will be released with a separate apk. You can click here to pre-download and install the test server. Please understand that the test is only available on Android devices due to the restrictions of Apple policy.
Please note that the accounts and data on Test Server are temporary, and are not related to your existing accounts on the Normal Server, and all data will be wiped after the test. Please try to avoid using the account and password on the test server that you frequently use.
To make sure every Pilot can better experience the updates and changes on Test Server, we will provide all needed resources like ISK, Skill Points and PLEX for you through in-game mail. Thank you for your understanding and continued support! All your suggestions and feedback are important to us!
Feedback entry: https://docs.google.com/forms/d/1_sseoleL2hmbJkTioAmALR79Vyx_K-oVE8upxgYm9bk/edit
 

New Features
Combat Commedation
Capsuleers can apply to the CONCORD for an additional reward for each bounty gained from repelling Pirates in the last 7 days (open the Wallet page> Filter > Bounty Rewards). CONCORD will grant ISK equal to 100% of the Bounty as an additional reward to encourage Capsuleers to continue fighting Pirates and defending the peace of New Eden.
Each bounty can only be claimed once, and it would be better for Capsuleers to destroy pirates as quickly as possible to get better results and claim extra bounties. Also, there is a max limit for each bounty of 200 million for now.
The higher the Tech Level, the more times you can apply for combat commendation every week. Unused applications can be stored for up to 4 weeks for future use. Please check the detailed rules in the in-game wallet interface.


Lazarus Unit Gift
Considering that Capsuleers may have Lazarus Unit needs after the balance adjustment, the CONCORD will provide Capsuleers with Lazarus Unit gifts.
During the event, when the Tech Level reaches 5, you can receive the Lazarus Unit. For higher Tech Levels, each level will unlock more rewards to be claimed. The exact amount is in accordance with the in-game event page.


Corporation Management Updates
When the Corporationleader is offline for more than 21 days, Corporation members can initiate a vote to change the CEO. Voting lasts for 48 hours. The vote is a registered ballot and the results will be made public. When voting is over, the Capsuleer with the highest number of votes will become the new Corps CEO.
Only Corporation members who have joined the Corporation and logged into the game within 3 days before the voting starts will be eligible to vote and can be voted for.


Anti-Drone Pulse Bomb
Introducing the Anti-Drone Pulse Bomb, a High Slot module that is usually fitted onto battleships. When enabled, it can damage all drones and fighters within a certain range.
The new module can be obtained from MK7 Module Blueprint Supply Box/MK9 Module Blueprint Supply Box.


PLEX Price Restrictions
A measure to stop malicious manipulation of the PLEX price. From now on, whenever a player attempts to sell PLEX/alter PLEX price/acquire PLEX, the price they set must be within this range:
Weekly Valuation x 80% ≤ Price ≤ Weekly Valuation x 120%
