Patch Notes - June 29

2022-06-28

Dear pilots of New Eden,
EVE Echoes will undergo maintenance on 6/29/2022 at 08:00 (UTC+0) and will take approximately 120 minutes to complete. During this time, players will be unable to log in to the game. We kindly ask for your understanding and patience during the maintenance.

New Content
CONCORD Pass
Duration: July 1, 2022 - August 12, 2022
CONCORD Pass Season 10 will start on July 1, 2022. This update will bring in the following content:
Imperial Battlecruiser Guardian Exile Nanocore Supply, which offers Prophecy Exile Core, Ferox Exile Core, Myrmidon Exile Core, and Cyclone Exile Core for your choice.
Please check the new CONCORD Pass for more information.

List System
The List System is designed to provide Capsuleers with tools to plan their manufacturing and manage items, especially for industrial players and corporation management.
The List System provides the following functions:
Through the List System, Capsuleers can select the specified number of items in the Inventory with one tap, as well as purchase the specified number of items with one tap.
Capsuleers can merge, select duplicates, double, and deduct the list.
Capsuleers can generate list from Industry, Contract, Blueprint, Inventory, Ship, Fitting, and the Combat Log systems.
Capsuleers can share the list via the Chat Channels, or copy it to the clipboard.

Corporation News
Added the Corporation News feature to the Corporation screen. Capsuleers can learn about the latest events of the corporation through the news.

Fleet Command
Added features: Standby Statistics, Commander Mode, Caretaker Mode and Apply to be a Commander
Standby Statistics
Upon initiation from the Commander, fleet members will choose to respond according to the ship types they pilot. The fleet panel will display the number of members who responded and ship types.
Commander Mode
Upon activation, all commands coming from the Fleet Commander will turn into fleet commands.
Caretaker Mode
Upon activation, fleet members will automatically respond to three fleet commands: Approach, Orbit and Stargate Jump after 8 seconds.
Apply to be a Commander
If the fleet Commander has not initiated any fleet command within 3 minutes, fleet members can select "Apply to be a Commander" in the fleet setting panel. Fleet Commander and Fleet Admiral will receive the application first. The application will be passed if one of the two permits; the application will fail if one of them rejects. If they disagree with each other, whoever made the first choice will get to decide. If they don't respond within 10s, the application will go to voting and achieve success once more than two-thirds (rounded down) of the fleet members agree to the application (no rejection means yes).

Optimizations
1. Updated estimated prices based on successful transactions in the Market.
2. The expiration period of planetary resource containers will be reset to 24 hours after server maintenance.
3. When editing orders in the market, you can change the price to the same as the original.
4. After Personal Citadels were unanchored, some players could not see obtainable containers simultaneously with other players because of network differences. A 5-minute unobtainable period has now been added to ensure that players with a non-optimized Internet connection also have an equal chance at these containers. The previous owner will be notified through mail after the other player obtains the Personal Citadel.
5. Added: Fuel Tank for Rorqual

Bug Fixes
1. Fixed the issue where Capsuleers couldn't initiate Regional Delivery in the Trading Center in Personal Assets.
2. Fixed the issue where Capsuleers docking at outposts couldn't sell items located in other stations on the Personal Assets page.
3. Fixed the issue where the effective duration of PLEX futures orders was incorrect.
4. Fixed the issue where the display of the Rorqual Shield Special Effect was too small.
