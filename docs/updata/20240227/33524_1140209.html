Patch Notes 0228

2024-02-27

Dear pilots of New Eden,

EVE Echoes will undergo maintenance on 2/28/2024 at 08:00 (UTC+0) and will take approximately 120 minutes to complete. During this time, players will be unable to log in to the game. We kindly ask for your understanding and patience during the maintenance.

 

Optimizations

Valuation updated based on the market transaction record.

Optimized the display effect of the Dragon Chant series SKINs under shadows.

 

Bug Fixes

Fixed a rare black screen issue after disconnecting and reconnecting to the game when piloting Rorqual.

Fixed an issue where the Insurance Points purchase interface might show fewer than 7 points.

Fixed a display issue with some SKINs on the Versatile Assault Ship.

Fixed the lighting display issue with the Neon Rain SKINs on some Carriers.

Fixed an issue where CyanSea's engine trail effect wasn't fully rendered.

Fixed an issue where Hubris might not be rendered in full or cause a crash in certain scenes.

Fixed a display issue with the facing of some turrets on Karura.

Fixed an issue where the weapon mount position was inaccurate for some Stealth Bombers.

Fixed a display issue that could occur when a ship decloaks in observer mode.

Fixed an issue where the remote repair interference ratio in Showdown Mode II was higher than the value shown.

Fixed an issue where the ship filtering feature might malfunction under certain filter criteria.

Fixed an issue where it is not possible to scroll the drone attribute in the Learning Protocol Library on some devices.
